# 🏰 Vertical Siege 3D (멀티 레이어 타워 디펜스)

> **Unity 6**를 활용한 3D 실시간 전략 타워 디펜스 게임.

## 🎮 프로젝트 정보
- **개발 기간**: 2026.04 ~ 2026.10 (예정)
- **개발 환경**: Unity 6 (6000.0.41f1)
- **사용 언어**: C#
- **IDE**: Visual Studio Code
- **타겟 플랫폼**: Mobile (Android)

## 🕹️ 핵심 규칙
1. **멀티 레이어 시스템**: 1층, 2층 등 여러 층에서 동시에 적들이 몰려옵니다.
2. **타워 파괴 및 잠금**: 타워가 파괴되면 수리 전까지 다음 웨이브 진행이 불가합니다.
3. **가챠 시스템**: 다양한 성능의 성벽과 수비병을 모집할 수 있습니다.
